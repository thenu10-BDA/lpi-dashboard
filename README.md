# 📊 Logistics Performance Dashboard

## 📌 Project Overview  
This project presents an interactive dashboard developed using **Streamlit** to analyse the **Logistics Performance Index (LPI)** dataset from the **World Bank**.  

The dashboard allows users to explore logistics performance across different countries and years through visualisations and interactive filters.

---

## 🎯 Objectives  
- Analyse logistics performance across countries  
- Identify top-performing countries  
- Compare multiple countries over time  
- Observe global trends in logistics performance  
- Provide simple and meaningful insights using data visualisation  

---

## 📂 Dataset  
- **Name:** Logistics Performance Index (LPI)  
- **Source:** World Bank Open Data  
- **Variables Used:** Country, Year, LPI Score  

---

## ⚙️ Features  
- 📅 Year selection filter  
- 🌍 Country comparison (multi-select)  
- 📊 Top 10 countries bar chart  
- 📈 Trend analysis over time  
- 📌 Key insights generation  
- 📋 Dataset preview  

---

## 🛠️ Technologies Used  
- Python  
- Pandas  
- Streamlit
